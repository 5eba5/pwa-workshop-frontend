Welcome to the Angular PWA!

This is the MVP progressive web app. Could be better!

Minimum requirements for the web app to call it a progressive one. A PWA should be:

  1. Served via HTTPS. The only exception is - localhost, for the development time
  2. Fast, responsive, mobile-first
  3. Available offline (at least the main UI)
  4. Installable on the OS / Homescreen

Nice to have:

  1. Push notifications
  2. Update flow
